# Claude 共用設定（所有 Example 專案適用）

## 使用者背景
- 使用者：Irene，大學教師
- 工作領域：教學網站與教育應用程式開發
- 主要受眾：大學部或高中學生

## 語言規則
- 所有回應、說明文字、文件內容 → 繁體中文
- 程式碼識別符（變數、函數、class 名稱）→ 英文（kebab-case 或 camelCase）
- 不得使用簡體中文或混用術語

## 術語標準化
| 術語 | 使用 | 不使用 |
|------|------|--------|
| 賽局理論 | ✓ | 博弈論 |
| 納許均衡 | ✓ | 納什均衡 |
| 帕累托最優 | ✓ | 帕雷托最優 |
| 優勢策略 | ✓ | 占優策略 |
| 報酬矩陣 | ✓ | 支付矩陣 |

## 回應風格
- 簡潔明確，不冗言贅述
- 程式碼修改後說明「為什麼」而非只描述「做了什麼」
- 適合教學情境的解釋深度（概念正確，避免過度技術化）

## 通用開發規範
- 優先編輯現有檔案，非必要不新增
- 不加超出需求的功能或抽象層
- 所有 HTML `<table>` 的 `<th>` 必須有 `scope="col"` 或 `scope="row"`
- 互動元素必須有 `aria-label` 或可辨識的文字標籤
- 避免在 `file://` 協定下使用 ES modules（用 IIFE 替代）

## 專案清單
- `website_example/` — 賽局理論靜態教學網站
- `app_example/` — 賽局理論互動 App（待開發）
