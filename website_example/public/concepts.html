參與者、策略與報酬

賽局理論的三個基本要素：

👥

參與者（Players）

賽局中做出決策的個體，可以是個人、公司、國家，甚至是基因。

🎯

策略（Strategies）

每個參與者在不同情境下的完整行動計畫，包含純策略與混合策略。

💰

報酬（Payoffs）

每種策略組合下，各參與者獲得的效用或收益值。

納許均衡

「在納許均衡中，給定其他人的策略，沒有任何玩家能夠透過單方面改變自己的策略來提高報酬。」 — John Nash, 1950

納許均衡是賽局理論最重要的解概念。一個策略組合 (s₁*, s₂*, ..., sₙ*) 是納許均衡，當且僅當對每個玩家 i：

ui(si*, s-i*) ≥ ui(si, s-i*)   對所有 si ∈ Si

重要性質： 每個有限賽局至少存在一個納許均衡（可能是混合策略均衡）。

優勢策略

若某策略無論對手如何行動，都能帶來比其他策略更高（或相等）的報酬，稱為優勢策略（Dominant Strategy）。

類型 定義 強度
嚴格優勢策略 無論對手選什麼，此策略報酬嚴格大於其他策略 強
弱優勢策略 無論對手選什麼，此策略報酬大於等於其他策略 弱
被優勢策略 存在另一策略在所有情況下都更好，此策略永遠不應被選 劣

最小最大值定理

在二人零和賽局中，馮·諾伊曼的最小最大值定理（Minimax Theorem）保證：

maxi minj u(i, j) = minj maxi u(i, j) = v*

玩家 A 最大化其最小報酬，玩家 B 最小化玩家 A 的最大報酬，二者在混合策略下必然達到同一個均衡值 v*。

注意： 此定理只適用於零和賽局。非零和賽局需要使用納許均衡概念。

帕累托最優

一個結果是帕累托最優（Pareto Optimal），若不存在另一個結果能讓至少一個參與者更好，同時又不讓任何人更差。

帕累托最優 ✓
(3,3)
非帕累托最優 ✗
(0,0)

囚徒困境中，納許均衡 (0,0) 並非帕累托最優，這正是「社會困境」的核心矛盾。

零和賽局

在零和賽局（Zero-Sum Game）中，所有參與者的報酬總和永遠為零：一人的收益必為他人的損失。

零和賽局例子

  • 西洋棋、圍棋
  • 撲克牌競技
  • 期貨市場（短期）
  • 軍事對峙

非零和賽局例子

  • 貿易談判
  • 公共財賽局
  • 囚徒困境
  • 協調賽局

混合策略

混合策略（Mixed Strategy）是指玩家依照某個機率分布隨機選擇純策略。

σi = (p₁, p₂, ..., pₖ)   其中 Σpⱼ = 1, pⱼ ≥ 0

為何使用混合策略？ 有些賽局沒有純策略納許均衡（如猜拳），但透過混合策略，每個有限賽局都保證存在至少一個均衡。

猜拳的均衡混合策略

每種手勢各以 1/3 的機率出現 → 對手無法藉由偏離來提高勝率
