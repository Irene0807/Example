# 語言規則

## 使用繁體中文
- 所有 HTML 頁面內容、標題、說明文字 → 繁體中文
- CSS class 名稱、JS 變數名稱 → 英文（小寫 kebab-case）
- 程式碼註解 → 英文或繁體中文皆可
- CLAUDE.md、README.md → 繁體中文

## 術語標準化
| 術語 | 使用 | 不使用 |
|------|------|--------|
| 賽局理論 | ✓ | 博弈論 |
| 納許均衡 | ✓ | 納什均衡 |
| 帕累托最優 | ✓ | 帕雷托最優 |
| 優勢策略 | ✓ | 占優策略 |
| 報酬矩陣 | ✓ | 支付矩陣 |

## 數字與標點
- 使用全形標點符號：，。！？；：「」
- 數字混合文字時：「3 個玩家」（數字前後各一空格）
