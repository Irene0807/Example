# 內容準確性規則

## 賽局理論定義
在新增或修改賽局理論內容時，必須符合以下標準：

### 納許均衡
- 定義：給定其他玩家策略，任何玩家單方面改變自身策略都不能提高報酬
- 一個有限賽局**必定存在**至少一個納許均衡（含混合策略）

### 報酬矩陣
- 格式：(玩家A報酬, 玩家B報酬)
- 納許均衡格應標記（★ 或特殊 CSS class `cell-nash`）
- 帕累托最優結果應說明

### 禁止事項
- 不得將「納許均衡」與「最優解」混用（均衡不一定是最優）
- 不得省略「帕累托最優」的概念說明
- 囚徒困境必須說明「個人理性 ≠ 集體理性」的核心矛盾

## HTML 無障礙規則
- 所有 `<table>` 的 `<th>` 必須有 `scope="col"` 或 `scope="row"`
- 互動元素必須有 `aria-label` 或可辨識的文字
